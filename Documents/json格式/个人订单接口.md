# 个人订单接口
## 前端请求
请求接口： /users/order
## 后端返回
### 普通用户
```json
{
    "code": 0,
    "message": "查看成功", 
    "data": {"Product": Product}
}
```
Product是一个数组，其中传值标准示例：
```json
{
    "product.name" : "1", 
    "product.photo" : "https://cdn.cnbj1.fds.api.mi-img.com/user-avatar/7b24d943-43c5-4744-9e2f-836faee06278.jpg" ,
    "product.address" : "山东省青岛市即墨区山东大学青岛校区",
    "product.quantity" : "1",
    "product.amount" : "100",
    "product.status" : 1,
    "product.shop_name" : "并夕夕旗舰店",
    "product.order_id" : "1" 
}
```
其中product.status为
* 0：放入购物车
* 1：已付款未发货
* 2：已发货未送达
* 3：已送达
### 商家
```json
{
    "code": 0,
    "message": "查看成功", 
    "data": {"Product": Product, "Shop": Shop}
}
```
