# 个人信息接口
## 获取个人信息
### 前端请求
请求接口：/users/information
```json
{}
```
其中，type为0代表顾客，type为1代表商家
### 后端返回
```json
{
    "code":0,
    "message":"Success",
    "data":{
        "phone":"15000000000",
        "id_card_number":"370900000000000000",
        "address":"山东省青岛市即墨区山东大学青岛校区",
        "photo":"https://cdn.cnbj1.fds.api.mi-img.com/user-avatar/7b24d943-43c5-4744-9e2f-836faee06278.jpg"
    }
}
```
## 修改个人信息
### 前端请求
请求接口：/users/edit
```json
{
    "phone":null,
    "id_card_number":"370900000000000000",
    "address":"山东省青岛市即墨区山东大学青岛校区",
    "photo":"暂无，等待实现"
}
```
不修改置为null
### 后端返回
```json
{
    "code":0,
    "message":"修改成功   ",
    "data":{
        "phone":"15000000000",
        "id_card_number":"370900000000000000",
        "address":"山东省青岛市即墨区山东大学青岛校区",
        "photo":"https://cdn.cnbj1.fds.api.mi-img.com/user-avatar/7b24d943-43c5-4744-9e2f-836faee06278.jpg"
    }
}
```
