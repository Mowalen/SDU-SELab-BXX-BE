## 发送对话消息
后端地址：/products/dialog

表单数据：

```json
{
    dialog: str = None
    receive_id: int = 0
}
```
## 接收对话消息

后端地址：/products/get_dialog

请求参数：send_id: int

返回信息：
0代表对面发的，1是自己发的

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "dialog": [
      {
        "status": 1,
        "content": "trhth",
        "create_dt": "2024-06-02T16:37:44"
      },
      {
        "status": 0,
        "content": "string",
        "create_dt": "2024-06-02T16:45:33"
      }
    ]
  }
}
```
