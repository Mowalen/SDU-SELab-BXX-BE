## 发送对话消息
后端地址：/products/dialog

表单数据：

```json
{
    dialog: str = None
    receive_id: int = 0
}
```
## 接收对话消息

后端地址：/products/get_dialog

请求参数：
```json
{
    "send_id" :int
}
```

返回信息：
0代表对面发的，1是自己发的

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "dialog": [
      {
        "status": 1,
        "content": "trhth",
        "create_dt": "2024-06-02T16:37:44"
      },
      {
        "status": 0,
        "content": "string",
        "create_dt": "2024-06-02T16:45:33"
      }
    ]
  }
}
```

## 查询与商家对话的所有用户

后端地址：/products/get_shop_dialog

请求数据：
```json
{
    "shop_id": 5
}
```

返回信息：用户user_id列表

```json
{
  "code": 0,
  "message": "OK",
  "data": [
    {
      "username": "2",
      "user_id": 2
    },
    {
      "username": "3",
      "user_id": 3
    }
  ]
}
```
