# 商家修改商品信息接口
## 前端请求
请求接口： 
| 字段名 | 值 |
| ----- | ----- |
| product_id | 123 |
| shop_id | 123 |
| image | FILE or null |
| description | "商品描述" or null |
| price | 1999.00 or null |
| name | "商品名" or null |
| stock | 1000 or null |
## 后端返回
```json
{
    "code": 0,
    "message": "修改成功", 
    "data": {}
}
```
