```json
{
    "Shopname" : name,
    "owner_id" : user_id,
    "owner_name":owner_name,
} 
```

```
{
    //返回成功或者失败信息
}
```

## 删除商店
接口地址：/shop/detial
### 前端请求
接口地址：
```json
{
    "shop_id" :"123"
}
```
### 后端返回
```json
{
    "code":0,
    "message":"成功",
    "data":{}
}
```



## 搜索页
接口地址：/shop/search
### 前端请求
```json
{
    "search_str" : "1234"
}
```
### 返回
```json
{
    "code":0,
    "message":"成功",
    "data":{
        "content":[
            {
                "Shopname" : "name",
                "owner_id" : "user_id",
                "owner_name":"owner_name", 
                "create_time" : "creation_time",
                "sales_volume" : "sales_volume",
            }
        ]
    }
}
```
