
### 前端请求
路径:/products/detail/change

```json
{
    product_id : int
    name:  None
    description:  None
    price: None
    category: None
    shop_id: None
    stock: None
    image: None
// 除id之外，均可空，**图片不改**
}
```

### 后端返回

#### 成功
```json


{
    "code":0,
    "message":"Success",
    "data":{
        "code":0
    }
}

```
#### 失败
```json


{
    "code":0,
    "message":"Success",
    "data":{
        "code":1
    }
}

```
