# 商店相关接口
## 新建商店
### 前端请求
接口地址：
```json
{
    "Shopname" :"name",
    "owner_id" : "user_id", 
    "owner_name":"owner_name",
}
```
### 后端返回
```json
{
    "code":0,
    "message":"成功",
    "data":{}
}
```
## 详情页
### 前端请求
接口地址：
```json
{
    "shop_id":"123"
}
```
### 后端返回
```json
{
    "code":0,
    "message":"成功",
    "data":{
        "Shopname" : "name",
        "owner_id" : "user_id", 
        "owner_name":"owner_name",
        "create_time" : "creation_time",
        "sales_volume" : "sales_volume",

        //商店的商品
        "product" :
        [
            {
                "url":
                "name":
                "price"
                "stock"
            
            },
            {
                "url":
                "name"
                "price"
                "stock"
            
            }
        ],

    }
}
```
