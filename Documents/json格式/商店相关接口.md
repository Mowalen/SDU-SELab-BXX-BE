#### 新建商店

```
{
            "Shopname" : name,
            "owner_id" : user_id, 
            "owner_name":owner_name,
}
```

```
{

}

```


### 详情页
#### 请求
```
{
    "shop_id" : id 
}
```
```
{
            "Shopname" : name,
            "owner_id" : user_id, 
            "owner_name":owner_name,
            "create_time" : creation_time,
            "sales_volume" : sales_volume,
} 
```

```

{
    //返回成功或者失败信息
```

### 搜索页

```
}
```

```
            "owner_id" : user_id
            "owner_name":owner_name, 
            "create_time" : creation_time,
            "sales_volume" : sales_volume,
```
