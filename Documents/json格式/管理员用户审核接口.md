
# 审核通过

## 前端请求

请求接口： /manager/allow_user

```json
{
    "shop_id":123
}
```

product_id类型为int

## 后端返回

```json
{
  "code": 0,
  "message": "修改成功",
  "data": 0
}
```


# 审核不通过

## 前端请求

请求接口： /manager/unallow_user

```json
{
    "shop_id":123
}
```

product_id类型为int

## 后端返回

```json
{
  "code": 0,
  "message": "修改成功",
  "data": 1
}
```
